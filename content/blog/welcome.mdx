## Why MDX

Posts here are authored as MDX files, which means I can drop React components straight
into the prose when a diagram or interactive widget makes the point better than words.

Code blocks support **line numbers**, **line highlights**, and **inline word highlights**:

```ts showLineNumbers {2} /name/
// even fenced code blocks render nicely
export const hello = (name: string) => `Hi, ${name}!`;

hello("world");
```
